Using PEZ: Edit and Delete Functions

Edit Use:

The Edit Options allow you to change a pole or zero's coordinates by hand. To access the edit options, press the "Edit by Co-Ord" button. The following menu will appear in the Main Control Window:


The Procedure for changing values using this menu is:



  • Press the "Select a Pole or Zero" Button

  • Move the mouse over to the Z-Plane and click on a pole or zero that you want to change.

  • Once an object has been selected, its co-ordinates will be placed in the edit boxes.

  • Make changes in the edit boxes as needed. Make sure you hit ENTER after typing a value in an edit box.

  • When all of the changes have been made to the selected, press the "Make Current Changes" button to update the object.

  • When Done editing objects, press the "Done Editing" button to return to the main menu.



      Note!

      The ANGLE edit box expects a numerical value in RADIANS/S, not Degrees.
      (This is expected to be changed in future vesrions of PEZ)

      Delete Use:

      The Delete options allow two different ways of deleting poles and zeros: single or multiple. The "Select & Delete Weighted" button will allow you to delete a single pole or zero with the mouse. The number of poles or zeros that are removed in cases where the selected object has an order higher than one depends on what the current weight is.

      The "Select & Delete Multiple" button does the same work as the "Select & Delete Weighted" button except that it allows for batch deletions. This allows the user to delete several objects in a row without having to click the "Select & Delete Weighted" button several times. To get out of the Delete Multiple mode, click the mouse outside of the Z-Plane.
