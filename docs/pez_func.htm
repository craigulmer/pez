PeZ Functionality

Starting PeZ

The booting procedure is as follows:
  1. Start Matlab
  2. Start PeZ in the Matlab window by cd'ing to the pez directory, and typing 'pez' (Note that v5.x of Matlab may have one warning message)
  3. Start filtdemo if desired
Notes:
  • PeZ has been tested with Matlab 4.2c-5.1, on Macs, PC's, and Unix (HP's and Suns).
  • GUI's look different from machine to machine (and I'm sorry for Win95)
  • Only one version of PeZ may be run at a time for each Matlab shell (because of global vars)

PeZ Windows

There are two figure windows to PeZ: the main control window, the plot window, and the configuration window.
 

Main Control Window

The Control Window is the below figure that does all of the filter editing work for PeZ.  On the left hand side there is a complex Z-Plane where poles and zeros are maintained and edited. The objects in this Z-Plane are all user controllable: you may select a pole or zero and simply drag it from its current location to a new destination. If the Real Time Drag Plots checkbox is selected, the updates in the plot window are made as the user drags the pole/zero around in the Z-Plane. Basic help information appears in the lower right hand corner for various operations.
 

The remaining buttons are documented below.
 

Add One-At-A-Time Edit by Co-Ordinate
Add Multiple Filter Gain
Add Double Import Filter Data
Multiplicity Export Filter Data
Select Delete
 

Plot Window

The Plot window display all of the filter's responses.
 
 
The checkboxes at the top of the window allow the user to change the magnitude to dB and the phase to Group Delay.

Printing: Due to the differences in printing between the versions of Matlab and OS's, the OS version of the print commands have been used as much as possible. That means that if there is not a print command naturally found in a figure window, PeZ adds one. However, most printing routines include the gui menu bar at the top of the figure window. Therefore, on most machines the user has an option to hide the gui bar by looking under the <PEZ Display Ops> menu option.

Pez Configuration Window

The Pez Configuration Window allows the user to specify settings, including bootup defaults.
 
 
  • Axis Mirroring: These checkboxes determine how PeZ mirrors poles and zeros in the Z-Plane. The default is to mirror vertically since the conjugates result in real polynomial filter coefficients.
  • Adding Precision: For any operation where a pole/zero is placed close to another pole/zero, PeZ checks a small radius to see if there are any other objects close by. If it finds an object, it assumes the user meant to place the new pole/zero on top of the old object. By increasing the adding precision, you decrease the search range for the add, making close objects more possible.
  • PeZ Boot Defaults: These defaults are saved in a file so that PeZ will boot to the specified configuration.
